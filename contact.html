聯絡我們

為避免你在 Windows / 手機上點信箱時跳出系統郵件 App（像你截圖那種畫面），BookWide 改用「客服頁」集中提供聯絡方式。

客服信箱

請寄信到：support@bookwide.net

用 Gmail 寫信 用 Outlook 寫信
✅ 推薦做法：用「Gmail / Outlook 網頁版」撰寫（不會跳出系統郵件 App）。
若你真的想用 mailto，可自行在瀏覽器設定預設 mail handler。
LINE（建議）

如果你已經有 LINE 官方帳號或邀請連結，放在這裡最直覺。

用 LINE 聯絡
若你還沒設定 LINE 連結：先用預留值 @YOUR_LINE_ID，之後再改。
你剛問的 A / B / C（建議順序）

A：直接 mailto（最省事，但常會跳出 Windows 郵件 App）

B：導到 Gmail/Outlook 網頁版撰寫（不跳 App，最好用）

C：做一個 contact.html（你現在這頁：把 B + LINE + 常見問題集中，最穩）

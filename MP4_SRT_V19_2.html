故事影片（MP4+SRT）→ Kling 動畫提示 JSON

流程：匯入源頭 MP4 + SRT → 依 SRT 自然分段（對齊 OK02）→ 自動截圖 → 產出每段 Kling 提示 → 匯出 JSON。
新功能：自動判斷動態層級（Calm/Medium/Tension）＋ 全域/每段「強度滑桿」可調，最終 Prompt 即時更新並與 JSON 一致。

請求下載支援的瀏覽器

使用服務時

在 BookWide 上開始學習時，需要使用「Google Chrome」、「Firefox」、「Microsoft Edge」或「Safari」。

使用非對應的瀏覽器（例如舊版瀏覽器）時，請下載上述任一瀏覽器後再開啟 BookWide。

※ 已經下載過「Google Chrome / Firefox / Microsoft Edge / Safari」任意一種的情況下無須再次下載。

3
點擊搜尋結果進入 BookWide，開始學習。

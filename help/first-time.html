第一次使用指南

這頁給第一次使用 BookWide 的你：從登入到開始學習，照著做就能跑起來。

1) 登入 / 註冊

  1. 到 首頁。
  2. 使用 Email 註冊或登入。
  3. 若忘記密碼，使用「重設密碼」流程（依你的設定頁面為準）。

2) 選擇內容開始學習

  1. 從內容索引選一支影片（Story/Music/Movie/News/Pro）。
  2. 進入播放器後，可在右側分頁切換：字幕 / 測驗 / 單字 / AI 互動。
小技巧： 如果你想看三語字幕（EN/ZH/JA），請確認該影片對應的 cues-*.json 已上傳並格式正確。

3) 測驗與分數

  • 測驗作答後會產生分數，用於學習控制台與排行榜（如果你有開啟）。
  • 若你發現分數一直是 0，通常是「寫入資料表」或「欄位名稱」不一致造成。

4) 常見問題

  • 看到 404：代表對應檔案尚未上傳到 GitHub Pages（或檔名大小寫不一致）。
  • 影片無法播放：先檢查影片網址/Storage 路徑是否正確。
  • 字幕/題目讀不到：檢查 JSON 檔名與 slug 是否一致。

5) 客服聯絡

若仍無法解決，請寄信 support@bookwide.net，並附上：Email、影片 slug、錯誤截圖、發生時間。
